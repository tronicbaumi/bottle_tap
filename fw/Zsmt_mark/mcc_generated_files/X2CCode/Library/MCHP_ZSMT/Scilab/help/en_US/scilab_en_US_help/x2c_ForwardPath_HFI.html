x2c_ForwardPath_HFI

MHCP inverse Clarke Park SVM Merged Transform for ZSMT

Block image not available.

Description

Merged Inverse Park, Inverse Clark and Space Vector Modulation Park_inv: alpha = d*cos(theta) - q*sin(theta) beta = d*sin(theta) + q*cos(theta) Clark_inv: a = beta b = (-beta + 1.732*alpha)/2 c = (-beta - 1.732*alpha)/2 SVM: Optimize output for PWM Description: Transform from the two-axis rotating d-q frame to the two-axis stationary frame α-β. Transform from the stationary two-axis α-β frame to the stationary three-axis, 3-phase reference frame. Optimize output for PWM

Inports

  • Ud: d voltage axis in rotating system
  • Uq: q voltage axis in rotating system
  • UdHF: HF injection to the d axis
  • theta: Rotor position

Outports

  • Va: Phase A out
  • Vb: Phase B out
  • Vc: Phase C out
  • Valpha: Inverse Park Voltage output
  • Vbeta: Inverse Park Voltage output

Mask Parameters

  • HF_InjectionPolarity: Adding to Ud --> Non inverting Subtract from Ud --> Inverting

Implementations

  • FiP16: 16 Bit Fixed Point
  • FiP32: 32 Bit Fixed Point
  • Float32: 32 Bit Floating Point
