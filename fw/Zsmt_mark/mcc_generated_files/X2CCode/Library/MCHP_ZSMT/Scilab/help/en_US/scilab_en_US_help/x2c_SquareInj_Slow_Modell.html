x2c_SquareInj_Slow_Modell

SquareInj_Slow_Modell

Block image not available.

Description

BE AWARE: This Block needs the SquareInjection_Fast-Block to work properly!!

Inports

  • dphi: angle error input signal
  • phi0: initial rotor angle Value which is preloaded at rising edge of Enable signal..
  • n0: initial speed preloaded at rising edge of Enable signal
  • Enable: Enable == 0: Deactivation of PLL functionality; Angle-Output is set input phi0; SYNC-Output toggles between 0 and 1; Enable 0->1: Preload of integral parts of PI and uI Enable == 1: Activation of this block

Outports

  • phi: calculated electrical rotor angle
  • Sync: Sync Signal: For Synchronization of HF-Injection with SlowTaskISR. Synchronization is done in SquareInjection_Fast-Block.
  • speed:

Mask Parameters

  • n_max: Maximum (mechanical) speed
  • p: Number of pole pairs
  • ts_fact: Multiplication factor of base sampling time (in integer format)
  • I_max: Normalization Value of Currents
  • Jp: Moment of Inertia [kg*m^2]
  • Keps: Error Current due to HF-Injection into wrong direction
  • rho: placement of Observer pole. The greater the pole is chosen, estimation of angle is done more dynamically
  • method: method of discretization of used LTI systems (PI, I)
  • selSpeedProp: Select where to place the emphasis of the speed output: dynamic or noise

Implementations

  • FiP16: 16 Bit Fixed Point
